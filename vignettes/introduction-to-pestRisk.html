Introduction to pestRisk: Modelling Pest Invasion Risk

Meriem Choukri

2026-06-14

Overview

pestRisk implements a complete Species Distribution Modelling (SDM) workflow to predict the current invasion risk of agricultural pest species. The package covers every step from raw data download to publication-ready risk maps and automatic reports.

GBIF occurrences → clean → WorldClim → VIF selection
    → Random Forest + MaxEnt → evaluate → risk map → report

The three target species are:

Species Common name Crop impact
Spodoptera frugiperda Fall armyworm Maize, sorghum, rice
Tuta absoluta Tomato leafminer Tomato, pepper, potato
Ceratitis capitata Mediterranean fruit fly Citrus, stone fruits

africa_ext <- c(-20, 55, -40, 40)

clim <- download_worldclim(
  path   = "~/pestRisk_data",
  res    = "10",
  extent = africa_ext,
  vars   = c(1, 4, 5, 6, 12, 15)   # BIO1, BIO4, BIO5, BIO6, BIO12, BIO15
)
terra::plot(clim, main = names(clim))

Why these 6 variables?

Variable Ecological relevance for pest insects
BIO1 (Annual Mean Temp.) Determines metabolic rate and development speed
BIO4 (Temp. Seasonality) Controls overwinter survival
BIO5 (Max Temp. Hottest Month) Heat stress upper threshold
BIO6 (Min Temp. Coldest Month) Cold-kill threshold for populations
BIO12 (Annual Precip.) Habitat moisture and host plant quality
BIO15 (Precip. Seasonality) Drought stress and host phenology

Step 10 – New features: ensemble model & degree-day risk

10a. Ensemble suitability (RF + MaxEnt)

ensemble_suitability() combines both models, weighted by their AUC scores from Step 6.

mx_risk <- predict_risk_map(mx_out, clim_small, breaks = c(0.33, 0.66))
#>   Low (): 100%

rf_auc <- eval_out$metrics$AUC[eval_out$metrics$Model == "Random Forest"]
mx_auc <- eval_out$metrics$AUC[eval_out$metrics$Model == "MaxEnt"]

ens <- ensemble_suitability(
  rf_risk = risk,
  mx_risk = mx_risk,
  rf_auc  = rf_auc,
  mx_auc  = mx_auc
)

terra::plot(ens$ensemble_suitability,
            main = "Ensemble suitability (AUC-weighted RF + MaxEnt)")

10b. Thermal degree-day risk (generations per year)

degree_days_risk() converts WorldClim monthly mean temperature into an estimate of how many pest generations can be completed per year — a direct measure of invasion intensity, not just presence/absence.

tavg <- geodata::worldclim_global(var = "tavg", res = 10,
                                   path = "worldclim_data")
tavg_small <- terra::crop(tavg, terra::ext(study_ext))

dd <- degree_days_risk(
  clim_stack      = tavg_small,
  t_base          = 12.5,   # S. frugiperda lower threshold
  n_gen_threshold = 450
)

terra::plot(dd$n_generations,
            main = "Estimated generations/year – S. frugiperda")

Interpretation: Pixels with ≥ 3 generations/year represent zones where chemical control is likely to fail due to rapid pest reproduction — these areas require integrated pest management (IPM) strategies.

Complete one-liner workflow

library(pestRisk)

species    <- "Tuta absoluta"
study_ext  <- c(-20, 55, -40, 40)
data_path  <- "~/pestRisk_data"

# 1. Download
occ   <- download_gbif(species, limit = 1500, year = "2010,2023")
clim  <- download_worldclim(path = data_path, res = "10",
                              extent = study_ext)

# 2. Clean
clean <- clean_occurrences(occ)

# 3. Background
bg <- generate_background_points(clim, clean$data, n = 5000)

# 4. Predictors
prep <- prepare_predictors(clean$data, bg, clim)

# 5. Models
rf_model <- train_rf_model(prep$model_data)
mx_model <- train_maxent(prep$model_data)

# 6. Evaluate
eval <- evaluate_models(rf_model, mx_model,
                         model_names = c("RF","MaxEnt"))

# 7. Map
risk <- predict_risk_map(rf_model, clim)

# 8. Summary
ctry <- summarize_country_risk(risk)

# 9. Report
generate_report(species, clean$data, clean$report, eval, risk, ctry,
                rf_model, output_file = "tuta_absoluta_report.html")
